# Support

Use the repository's issue forms for reproducible bugs and feature proposals. Include the Skill Steward version, Node.js version, operating system, command or route, and sanitized output. Remove home directories, repository names, Skill bodies, credentials, and private Git URLs before posting.

For suspected vulnerabilities, follow [SECURITY.md](SECURITY.md) instead of opening a public issue. This community project provides best-effort support; no response-time or compatibility SLA is implied.
